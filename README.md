# pdf-paragraph-extractor
Extract paragraphs from PDF and EPUB file

## Features
- Upload and manage PDF & EPUB files
- Extract and copy paragraphs with ease
- Add notes for each page
- Text-to-speech functionality
- Search through paragraphs
- **Dark mode support** - Toggle between light and dark themes
- Sync data across devices (export/import)
